## About

This project is a demo blog website built to illustrate how a modern
GraphQL-powered front-end & back-end application can come together.

## Features

-   View list of blog posts
-   Read individual blog posts
-   Responsive UI
-   GraphQL data operations

## Tech Stack

-   TypeScript
-   React
-   Tailwind CSS
-   Vite
-   GraphQL API

## Getting Started

### Installation

    git clone https://github.com/mayank-207/blogs.git
    cd blogs
    npm install

### Run

    npm run dev

## Environment Configuration

Set up `.env` based on `.env.example`.
